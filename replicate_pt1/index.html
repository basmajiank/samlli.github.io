Directions You will, to the best of your ability, replicate this HTML page. No style commands and no CSS are used anywhere in this document.

You will need the <code> to get monospaced font during this project. You will also need the <pre> tag. You should think about why they are different.

Use the HTML5 Code validator, linked here to see to it that your HTML is clean. My elves will check to see if your code validates. Failure to validate carries a penalty.

Watch for little details! They lurk.

HTML Special Character Entities

To do this assignment you will need HTML character entities. They all look like this: &string;.

Table of Character Entities
EntityWhat AppearsName
&amp;&ampersand
&gt;>greater than
&lt;<lt
&copy;©copyright symbol
&sect;§section symbol

More of them can be found on the W3Schools website.

Here we now show the HTML code to make the table of character entities. Go ahead; hate me now, but if you are good at vi it's not hard.

Useful Acronyms at NCSSM

  • PFM This stands for "Premeditated Food Murder;" it's where you eat.
  • RLA This stands for "Residence Life Assistant," who catches you on hall at 3AM and gives you a level.
  • RTFM This stands for "Read the fun manual;" it's what your CS teacher tells you to do.
  • FMFBO This stands for "Freeze my fool butt off;" it's what happens every year if you go to school in boston.
  • PEC This stands for "Physical Education Center;" it's a place you should not avoid.
