Directions You will, to the best of your ability, replicate this HTML page. Name this file replicateThis2.html. Put your CSS in a file named two. css; it has been linked for you. To submit it, do this.

Submitting Begin by making sure your home direction has these two directories, submitted and returned. If not, make them as follows


    $ cd
    $ mkdir submitted returned

Then navigate to your HTML and CSS files and do this.


    $ cp replicateThis2.html ~/submitted
    $ cp two.css ~/submitted

Use the HTML5 Code validator, linked here to see to it that your HTML is clean. My elves will check to see if your code validates. Failure to validate carries a penalty.

Here is a fun new thing to do! Ever want to download stuff from the web without a browser? You can get images and source in a terminal with the command wget. Suppose I go to Gary Bishop's website and I see his picture in my browser. I right click on it and choose the option "copy image address." Now I do this in my terminal.
